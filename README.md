# 🟢 Netflix Clone - Imersão Front-End com IA

Este projeto é um clone da interface da Netflix, desenvolvido durante a **Imersão Front-End com IA da Alura**. O foco principal foi aprender a integrar o desenvolvimento web tradicional (HTML, CSS e JS) com ferramentas de Inteligência Artificial para acelerar o fluxo de trabalho e refatoração de código.

----------

## 🚀 Tecnologias Utilizadas

-   **Front-End:** HTML5, CSS3 e JavaScript (ES6+).
    
-   **IA para Devs:** ChatGPT, GitHub Copilot e Gemini (utilizados para lógica, sugestões de estilo e depuração).
  
    
-   **Versionamento:** Git e GitHub.
    

----------

## 🛠️ O que foi desenvolvido?

O projeto simula a experiência do usuário na plataforma de streaming, incluindo:

-   **Header Responsivo:** Menu de navegação que se adapta a diferentes tamanhos de tela.
    
-   **Banner Principal:** Destaque para filmes/séries com efeitos de transição.
    
-   **Carrossel de Filmes:** Listagem de títulos organizada por categorias (usando bibliotecas como Swiper.js ou Flexbox/Grid).
    
-   **Interatividade:** Efeitos de _hover_ nos cards de filmes e funcionalidade de scroll.
    

----------

## 🤖 Uso de Inteligência Artificial

Neste projeto, a IA não apenas ajudou a escrever código, mas foi usada para:

1.  **Refatoração:** Otimizar seletores CSS para tornar o código mais limpo.
    
2.  **Lógica de JS:** Criação de funções para manipulação do DOM e eventos de clique.
    
3.  **Acessibilidade:** Sugestões de tags ARIA e melhores práticas para leitores de tela.

## 🔧 Como rodar o projeto localmente

1.  No terminal (Git Bash), clone o repositório:
    
    Bash
    
    ```
    git clone https://github.com/brandao-vini/netflix-alura-front.git
    
    ```
    
2.  Entre na pasta do projeto:
    
    Bash
    
    ```
    cd netflix-alura-front
    
    ```
    
3.  Abra o arquivo `index.html` no seu navegador ou use a extensão **Live Server** do VS Code.
